Task 3:

You have to find the card number 2 again

Notice that we repeat the same procedure for each card

Therefore we will go through each card by a counter

Let i be a counter from 0 to 4

Check the i card. If it is 2, print the position

Increase i by 1

The process of going through each element of the sequence is called (one) iteration.

The keyword for iteration is: for or while

Iteration is another basic structure we will use for writting Algorithms
